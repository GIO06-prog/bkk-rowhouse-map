# แผนที่ความหนาแน่นตึกแถว กรุงเทพมหานคร พ.ศ. 2561

แผนที่ออนไลน์แสดงจำนวนอาคารประเภทตึกแถวรายเขตทั้ง 50 เขตของกรุงเทพมหานคร จัดทำจากฐานข้อมูลภูมิสารสนเทศ (GIS) เพื่อเผยแพร่ต่อสาธารณะ

**เปิดดูแผนที่:** https://gio06-prog.github.io/bkk-rowhouse-map/

## เกี่ยวกับโครงการ

แผนที่นี้แสดงการกระจายตัวของอาคารประเภทตึกแถวในพื้นที่กรุงเทพมหานคร รวมทั้งสิ้น 400,461 อาคาร จำแนกรายเขตและกลุ่มเขต ระบายสีตามระดับความหนาแน่น (choropleth) ผู้ใช้สามารถเลื่อนเมาส์หรือแตะที่เขตเพื่อดูจำนวนอาคาร กลุ่มเขต และอันดับความหนาแน่นได้

## ข้อมูลสำคัญ

| รายการ | รายละเอียด |
|--------|-----------|
| จำนวนอาคารตึกแถวทั้งหมด | 400,461 อาคาร |
| ขอบเขตพื้นที่ | 50 เขต / 6 กลุ่มเขต |
| ปีข้อมูล | พ.ศ. 2561 |
| เขตที่มีตึกแถวมากที่สุด | ธนบุรี (22,376 อาคาร) |
| เขตที่มีตึกแถวน้อยที่สุด | หนองจอก (1,238 อาคาร) |

## ที่มาของข้อมูล

- **ข้อมูลจำนวนตึกแถว:** ฐานข้อมูลภูมิสารสนเทศ (GIS) ประเภทตึกแถว สำนักงานภูมิสารสนเทศ สำนักการวางผังและพัฒนาเมือง กรุงเทพมหานคร (พ.ศ. 2561)
- **ขอบเขตการปกครอง:** ข้อมูลขอบเขตเขตกรุงเทพมหานคร (แปลงพิกัดเป็น WGS84 / EPSG:4326)
- **แผนที่ฐาน:** © OpenStreetMap contributors, © CARTO

## วิธีการจัดทำ

ข้อมูลขอบเขตระดับแขวงถูกรวม (dissolve) เป็นระดับเขต แปลงระบบพิกัดจาก UTM Zone 47N (EPSG:32647) เป็น WGS84 (EPSG:4326) และลดความละเอียดของรูปทรง (simplify) เพื่อให้ไฟล์มีขนาดเล็กและแสดงผลได้รวดเร็ว จากนั้นเชื่อมโยงกับข้อมูลจำนวนตึกแถวรายเขต แสดงผลด้วยไลบรารี Leaflet เป็นเว็บแบบ static

## โครงสร้างไฟล์
